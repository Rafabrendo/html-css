Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declara-lo no HTML, ou par selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto

Propriedades disponíveis:

  • color
  • background
  • font
  • Propriedades de texto

Estiliza o plano de fundo de elementos em modo de tela cheia, como dialog ou video
