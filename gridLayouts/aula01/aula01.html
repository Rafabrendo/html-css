fr é uma unidade de medida relativa e precisa de uma referencia. Nesse caso, pegou a referencia (width) da pagina toda.

Função repeat

Permite repetir para todos, ou para algumas das faixas, o mesmo valor para seu tamanho de coluna ou linha. ex.:

        
            
Antes:

            .container{
                grid-template-columns: 1fr 1fr 1fr;
            }
        
        
            
Depois:

            .container{
                grid-template-columns: repeat(3, 1fr);
            }

repeat(3, 1fr 2fr); -> mesma coisa que: 1fr 2fr 1fr 2fr 1fr 2fr
